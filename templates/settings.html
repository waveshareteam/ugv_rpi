For the first time use of this Pan-Tilt, you will need to follow these steps to config the servos' ID and middle position:

Tilt Servo ID: 1Pan Servo ID: 2

The default ID of the Tilt Servo and Pan Servo is both 1 at the beginning, but the ID of the Pan Servo should be set to 2

1

Disconnect the wire of the Tilt Servo, make sure you are setting the ID of Pan Servo.

2

Click the [Set Pan ID] button to configure the current servo as Pan Servo, which will set the ID of the Pan Servo as 2.

Set Pan ID
3

Then you can connect the wire of the Tilt Servo back to the Pan-Tilt.

4

Now we can begin to set the middle position and click the [Release] button to unlock the torque lock of the two servos.

Release
5

As shown in the picture below, manually adjust the angle of the two servos so that the camera is pointed straight ahead.

6

Click the [Middle Set] button to set the current position as the middle position.

Middle Set
7

Then you can begin to test the Pan-Tilt to turn to different angles. and check whether it can work normally.

8

if the Pan-Tilt is not operating normally, you may try to redo the whole process again.

Note: If you didn't disconnect the Tilt Servo in step 1, then both servo IDs will be set to 2 after you click the [Set Pan ID] button. In this case, you need to click [Set Tilt ID] to restore both servo IDs to 1, then repeat the entire setup process!
